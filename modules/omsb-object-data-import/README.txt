This module is used to store the Object values programtically by importing the excel file.

Steps to import data into liferay objects
1. Pass the Object External Reference Code into the textbox.
2. Create excel data
   a) Headers are the object field names
   b) Kepp the Header at first row always(As per the written code in this module).
   c) Prepare your data in the excel and upload in this portlet.
